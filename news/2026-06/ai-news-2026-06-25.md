## 模型发布

### 1. 阿里 Qwen-AgentWorld：七域统一语言世界模型

- <strong>日期：</strong>GitHub / Hugging Face 发布 2026-06-24；VentureBeat 报道 2026-06-24。
- <strong>事件：</strong>阿里 Qwen 团队开源 **Qwen-AgentWorld-35B-A3B**（MoE，35B 总参 / 3B 激活，256K 上下文）及 **AgentWorldBench**。模型不直接做 agent 动作，而是预测 agent 交互后的下一环境状态，覆盖 MCP、Search、Terminal、SWE、Android、Web、OS 七域。
- <strong>为什么重要：</strong>将「世界建模」作为 CPT 阶段起的原生训练目标，而非事后补丁；论文称在模拟环境中训练的 agent 可超越仅在真实环境训练的表现，并支持零样本迁移到 OOD 环境。
- <strong>核验状态：</strong>已用 GitHub README、arXiv 摘要、VentureBeat 交叉核验。
- <strong>来源：</strong>[QwenLM/Qwen-AgentWorld](https://github.com/QwenLM/Qwen-AgentWorld)，[arXiv:2606.24597](https://arxiv.org/abs/2606.24597)，[VentureBeat](https://venturebeat.com/technology/alibabas-model-never-trained-as-an-agent-and-improved-agent-performance-across-seven-benchmarks)

### 2. Google Gemini 3.5 Flash：computer use 成为内置工具

- <strong>日期：</strong>2026-06-24。
- <strong>事件：</strong>Google 官方宣布 computer use 从独立 Gemini 2.5 computer use 模型，升级为 **Gemini 3.5 Flash 内置工具**，可通过 Gemini API 与 Gemini Enterprise Agent Platform 调用；支持浏览器、移动端、桌面端 GUI 交互，并提供敏感操作确认、间接 prompt injection 自动中止等企业级安全选项。
- <strong>为什么重要：</strong>降低 agent 自动化开发门槛——开发者无需单独挂载 computer use 模型；与 OpenAI Codex、Claude computer use 形成直接产品竞争。
- <strong>核验状态：</strong>已用 Google 官方博客核验。
- <strong>来源：</strong>[Google Blog - Computer use in Gemini 3.5 Flash](https://blog.google/innovation-and-ai/models-and-research/gemini-models/introducing-computer-use-gemini-3-5-flash/)

- <strong>日期：</strong>2026-06-24。
- <strong>事件：</strong>OpenAI 与 Broadcom 发布 **Jalapeño** Intelligence Processor，专为大语言模型推理设计；工程样片已在实验室运行 GPT‑5.3‑Codex‑Spark 等工作负载。OpenAI 称早期测试显示每瓦性能显著优于当前先进水平，完整技术报告将于数月内发布；计划 2026 年底前开始部署，并与微软等伙伴扩展至千兆瓦级数据中心。

- <strong>核验状态：</strong>已用 OpenAI 官方公告、Broadcom 投资者新闻稿核验；50% 成本说法来自媒体报道引述高管，非正式技术报告。
- <strong>来源：</strong>[OpenAI - Jalapeño chip](https://openai.com/index/openai-broadcom-jalapeno-inference-chip/)，[Broadcom IR](https://investors.broadcom.com/news-releases/news-release-details/openai-and-broadcom-unveil-llm-optimized-intelligence-processor)，[TechCrunch](https://techcrunch.com/2026/06/24/openai-unveils-its-first-custom-chip-built-by-broadcom/)

## 论文与研究

### 1. Qwen-AgentWorld：通用 agent 的语言世界模型

- <strong>日期：</strong>arXiv 提交 2026-06（编号 2606.24597）。
- <strong>主题：</strong>提出 native language world model，通过 CPT → SFT → RL 三阶段训练，在七域 agent 环境中做下一状态预测；构建 AgentWorldBench，用 Claude Opus 4.6 等前沿模型真实轨迹做 OOD 评测。
- <strong>为什么重要：</strong>将 agent 训练从「在真实环境试错」部分转移到「高保真可控模拟器」，与 OpenAI Ona 收购、Google computer use 同属 agent 基础设施竞争。
- <strong>核验状态：</strong>已用 arXiv 摘要与 GitHub README 核验。
- <strong>来源：</strong>[arXiv:2606.24597](https://arxiv.org/abs/2606.24597)
