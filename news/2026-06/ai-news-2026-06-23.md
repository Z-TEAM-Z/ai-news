## 模型发布

### 1. OpenAI GPT‑5.5‑Cyber：从发现漏洞到自动补丁

- <strong>日期：</strong>2026-06-22。
- <strong>事件：</strong>OpenAI 在 Daybreak 公告中发布 GPT‑5.5‑Cyber 完整版，定位为面向已验证防御者的高级网络安全模型，更“permissive”且更强于漏洞发现、验证、补丁生成与证据整理。
- 关键数据：CyberGym 85.6%（对比 GPT‑5.5 的 81.8%）；ExploitGym 39.5% vs 25.95%；SEC-bench Pro 69.8% vs 63.1%。OpenAI 称这是其单模型在 CyberGym 上的最高成绩。
- <strong>为什么重要：</strong>AI 网络安全竞争正从“谁能发现更多漏洞”转向“谁能更快把发现变成可部署补丁”；这与同日 Five Eyes 声明、Patch the Planet 倡议形成同一叙事。
- <strong>核验状态：</strong>已用 OpenAI 官方博客核验。
- <strong>来源：</strong>[OpenAI - Daybreak: Tools for securing every organization in the world](https://openai.com/index/daybreak-securing-the-world/)

### 2. Sakana Fugu / Fugu Ultra：多智能体编排即“一个模型 API”

- <strong>日期：</strong>Sakana 于 2026-06-22 宣布；The Verge 报道时间为 2026-06-23 00:04 UTC。
- <strong>事件：</strong>日本 Sakana AI 推出 Fugu 多智能体编排系统，通过单一 OpenAI 兼容 API 在内部路由多个专家模型；旗舰档 Fugu Ultra（model ID `fugu-ultra-20260615`）面向高难研究、网络安全分析、专利/文献调查等任务。
- <strong>为什么重要：</strong>在 Anthropic Fable/Mythos 受美国出口管制影响后，Fugu 被定位为“单供应商依赖”的替代路线；本质是 orchestration layer，而非自研单一大模型。
- <strong>核验状态：</strong>已用 The Verge 与多家媒体报道交叉核验；Sakana 官方产品页/定价细节本轮未直接打开原文，benchmark 对比需结合 Sakana 自家披露谨慎看待。
- <strong>来源：</strong>[The Verge - Sakana Fugu Ultra](https://www.theverge.com/ai-artificial-intelligence/953904/sakana-fugu-ai)，[VentureBeat - Sakana Fugu](https://venturebeat.com/orchestration/no-claude-fable-5-no-problem-sakana-achieves-frontier-performance-with-new-fugu-multi-model-auto-synthesis-system)

## 论文与研究

### 1. OpenClaw-Skill：面向 agent 的 Collective Skill Tree Search

- <strong>日期：</strong>arXiv 编号 2606.16774（2026 年 6 月提交）。
- <strong>主题：</strong>论文提出 CSTS（Collective Skill Tree Search）与 CSRL（Collective Skill Reinforcement Learning），自动构建结构化、可迁移的 agent skill 树，并在 OpenClaw 风格环境中训练 OpenClaw-Skill 模型。
- <strong>为什么重要：</strong>随着 OpenClaw、deer-flow 等 agent harness 爆发，skill 如何自动发现、组合与跨模型迁移正成为研究热点。
- <strong>核验状态：</strong>已打开 arXiv 摘要页核验。
- <strong>来源：</strong>[arXiv:2606.16774](https://arxiv.org/abs/2606.16774)

### 2. AIDev 等近期 agent 数据集研究（延续关注）

- <strong>日期：</strong>arXiv 提交于 2026-02-09。
- <strong>主题：</strong>聚合 93 万+ agent-authored PR，覆盖 Codex、Devin、Copilot、Cursor、Claude Code 等五类 agent。
- <strong>为什么重要：</strong>AI 编程 agent 研究正从离线 benchmark 转向真实 PR/commit/review 数据；本轮无更新，但仍是理解 agent 工程落地的关键参考。
